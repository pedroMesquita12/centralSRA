Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) e tem como objetivo a criação de uma plataforma web para 
gerenciamento e contabilização de horas complementares de estudantes.

O sistema permite o registro, acompanhamento e organização das atividades realizadas pelos alunos, facilitando o controle das horas 
exigidas pela instituição de ensino. Além disso, a plataforma foi projetada para possibilitar a integração e o envio direto dessas 
informações à faculdade por meio dos programas educacionais do Centro Paula Souza, com foco nos cursos do modelo AMS.

A proposta busca otimizar processos acadêmicos, reduzir burocracias administrativas e oferecer maior praticidade tanto para estudantes quanto para a instituição.

 ##

This project was developed as a Final Course Project (Capstone Project) and aims to create a web platform for managing and tracking 
students’ complementary academic hours.

The system enables students to register, monitor, and organize their completed activities, making it easier to control 
the number of hours required by the educational institution. Additionally, the platform was designed to allow integration 
and direct submission of this information to the college through the educational programs of the Centro Paula Souza, with a focus on AMS courses.

The proposal seeks to optimize academic processes, reduce administrative bureaucracy, and provide greater convenience for both students and the institution.
